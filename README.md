# Funcionalidades principales

## Comprador

- Inicio de sesión
- Recuperación de contraseña
- Acceso biométrico
- Pantalla principal
- Catálogo de productos
- Detalle de producto
- Carrito de compras dinámico
- Pasarela de pagos dinámica
- Perfil de usuario

## Administrador

- Dashboard principal
- Gestión de usuarios
- Gestión de productos
- Reporte de ventas

## Vendedor

- Panel vendedor
- CRUD de productos
- Historial de pedidos

# Características visuales

- Diseño moderno y minimalista
- Colores personalizados
- Drawables personalizados
- Navegación funcional
- Interfaces responsive
- Consistencia visual entre pantallas

# Funcionalidades técnicas implementadas

- Navegación mediante Intents
- Uso de Toasts
- Uso de ScrollView y LinearLayouts
- Implementación de biometría Android
- Totales dinámicos en carrito y pagos
- Métodos de pago dinámicos
- Reutilización de drawables
- Personalización de themes y colors

# Instalación

1. Clonar el repositorio
2. Abrir el proyecto en Android Studio
3. Sincronizar Gradle
4. Ejecutar en dispositivo físico o emulador
